    # AI Dashboard

    Students edit React components to complete an AI dashboard page.

    ## Project Goal
    This project is already set up for students. They do **not** need to build the project from scratch.
    Their job is to open the React components and edit the JSX content.

    ## Student Tasks
    1. Update the Header component title and subtitle.
2. Add 3 AI tool cards in the ToolGrid component.
3. Edit the StatsPanel component to display 3 useful stats.
4. Customize the Footer component with course or student info.
5. Style one section with your own class names in App.css.

    ## Project Structure
    - `src/App.jsx` - main page layout
    - `src/App.css` - styles
    - `src/components/Header.jsx` - top section
    - `src/components/Footer.jsx` - bottom section
    - one or more feature components depending on the project

    ## How to Run
    1. Open a terminal in this project folder
    2. Run `npm install`
    3. Run `npm run dev`
    4. Open the local browser URL shown by Vite

    ## Student Focus
    - Edit component text
    - Add JSX elements
    - Update arrays and lists
    - Practice props and reusable components
    - Make small style changes
