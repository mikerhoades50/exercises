    # AI Prompt Lab

    Students edit React components to build a prompt practice page.

    ## Project Goal
    This project is already set up for students. They do **not** need to build the project from scratch.
    Their job is to open the React components and edit the JSX content.

    ## Student Tasks
    1. Update the PromptExamples component with 3 prompt examples.
2. Edit the Tips component to include 4 prompt writing tips.
3. Customize the Header component text.
4. Update the Footer component with your class name or bootcamp name.
5. Add one more JSX section inside App.jsx.

    ## Project Structure
    - `src/App.jsx` - main page layout
    - `src/App.css` - styles
    - `src/components/Header.jsx` - top section
    - `src/components/Footer.jsx` - bottom section
    - one or more feature components depending on the project

    ## How to Run
    1. Open a terminal in this project folder
    2. Run `npm install`
    3. Run `npm run dev`
    4. Open the local browser URL shown by Vite

    ## Student Focus
    - Edit component text
    - Add JSX elements
    - Update arrays and lists
    - Practice props and reusable components
    - Make small style changes
