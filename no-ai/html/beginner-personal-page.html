My Personal Web Page

This activity helps beginners practice the most common HTML tags.

Student Tasks

  1. Add an h2 heading called "About Me".
  2. Add a paragraph introducing yourself.
  3. Add an image of a hobby, favorite place, or favorite food.
  4. Add a list of 3 favorite activities.
  5. Add a link to your favorite website.

HTML lets you structure content on a web page using headings, paragraphs, lists, images, and links.
