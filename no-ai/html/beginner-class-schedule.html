Weekly Class Schedule

This activity helps students practice tables and page structure.

Student Tasks

  1. Add a table with columns for Day, Topic, and Homework.
  2. Include at least 5 rows for weekdays.
  3. Add a section called "Supplies Needed".
  4. Add an unordered list with at least 4 items.
  5. Add a contact email link at the bottom of the page.

Course Overview

In this course, students will learn beginner web development skills by building simple pages with HTML.
